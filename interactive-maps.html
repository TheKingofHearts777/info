Interactive Maps Application

This application I made using JavaScript, HTML, and CSS
It is a program intended to be used for tabletop role-playing
games such as Dungeons and Dragons or Pathfinder. It can be used
by the Game Master or player to keep notes on where certain locations
are or used as a path tracker to see where you've been.
