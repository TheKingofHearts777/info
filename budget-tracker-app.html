Budget Management Application

This project is a budget management application that I created using HTML, CSS, and JavaScript. The application allows users to input their income and expenses per month, and then calculates their total income, total expenses, and remaining budget per month. The application allows users to add multiple income and expense entries, and it updates the calculations in real-time as the user inputs data. The user can go to previous months and future months to see their budget for those months as well. All information is saved in the browser's local storage, so the user can close the application and come back to it later without losing their data. The application also includes a simple and clean user interface, making it easy for users to navigate and manage their budget effectively.
