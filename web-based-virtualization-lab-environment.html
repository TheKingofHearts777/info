Web-based Virtualization Lab Environment

This project was my Senior year capstone project at
college. I mainly worked on the backend server side
of the project, and helped out some with the database
which was MongoDB. The project's goal is to create a
website where CS/Cyber students like myself could
login to our program, go to their class page and complete
various labs that their teacher/professor has assigned
them that day or week. The project spans many disciplines
including Java backend, Angular JS frontend, VM Networking, Nginx,
and MongoDB database. We used Proxmox as our VM platform
which a teammate and I worked with extensively, trying to get the
VM's showing up in the browser. During out time with this
project, we weren't able to connect the VM's to the website.
